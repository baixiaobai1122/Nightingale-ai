## Healthcare Compliance Notes
- This software is designed with HIPAA compliance in mind
- All PHI (Protected Health Information) is redacted before AI processing
- Row Level Security ensures data isolation between users
- Audit trails are maintained for all medical data access
- Real patient data should never be used in development/testing
